# 오로라 주스 가게 — 어린이집 카페 키오스크

어린이집 아이들의 카페 놀이용 키오스크 웹페이지입니다.
`키오스크.pptx` 화면 설계서(13장)를 분석해 그대로 웹으로 옮겼습니다.

## 배포 주소 (GitHub Pages)

| 화면 | 주소 | 로그인 |
|---|---|---|
| 키오스크 (손님) | https://kocaruce.github.io/cafe-kiosk/ | 불필요 |
| 주문 현황 (직원) | https://kocaruce.github.io/cafe-kiosk/pos.html | 필요 🔒 |
| 주문 내역 (날짜별) | https://kocaruce.github.io/cafe-kiosk/history.html | 필요 🔒 |

직원 페이지는 Firebase 로그인 후에만 열립니다. 태블릿마다 최초 1회만 로그인하면 유지됩니다.

## 실행 방법

정적 웹페이지라서 아무 웹서버로 열면 됩니다.

```bash
npx -y serve -l 8765 .
# 브라우저에서 http://localhost:8765 접속
```

키오스크 태블릿/PC에서는 브라우저 전체화면(F11)으로 사용하세요.
화면은 1920×1080 기준으로 만들어졌고, 어떤 해상도든 비율에 맞춰 자동 축소/확대됩니다.

## 화면 흐름 (PPT 슬라이드 순서)

1. **시작 화면** — 아이가 그린 "오로라 주스 가게" 간판. 화면을 누르면 주문 시작
2. **주문 화면** — 카테고리 5개(주스 / 에이드 / 스무디 / 라떼 / 커피, 티), 메뉴를 누르면 장바구니에 담김
3. **주문 목록** — 수량 조절(+/−), 합계 확인 후 결제하기
4. **결제 화면** — 카드결제 / 현금결제 선택
5. **현금**: "현금은 직원에게 주세요." → 주문 완료
   **카드**: "카드를 넣어주세요." → "결제 중..." → "결제가 완료되었습니다." → 주문 완료
6. **주문 완료** — 몇 초 후 자동으로 시작 화면으로 돌아감

## 파일 구성

- `index.html` — 화면 9개 (시작/주문/주문목록/결제/안내 5종)
- `css/style.css` — PPT 색상 그대로 (배경 `#DFF7C2`, 포인트 `#53AB58`)
- `js/app.js` — 메뉴 데이터, 장바구니, 화면 전환, 효과음
- `assets/images/` — PPT에서 추출한 이미지 77개 (음료, 과일 장식, 아이콘)
  - `sign-title.png` — 아이들이 쓴 "오로라 주스 가게" 글씨 컷아웃 (시작 화면 간판)
  - `100won.png` — 100원 동전 (1,000원 미만 가격 표시용, 투명 배경)

## 메뉴와 가격

가격은 아이들이 손으로 쓴 **"오로라 주스 가게" 간판(최종 확정본)** 기준입니다.
키위망고 2,300원 · 레몬 2,100원 · 오렌지에이드 1,200원처럼 아이들이 정한
가격도 그대로 반영했고, 화면에는 천원 지폐 n장 + 나머지는 동전 1개로 표시됩니다.
가격 수정은 `js/app.js` 맨 위 `MENU` 객체에서 하면 됩니다.

## 폰트

PPT 원본은 "210 키위바나나", "Lazydog" 폰트를 사용합니다.
해당 폰트가 설치된 PC에서는 자동으로 적용되고, 없으면 시스템 고딕체로 표시됩니다.
